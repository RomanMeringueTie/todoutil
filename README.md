# todo - трекер задач в вашем проекте

## Использование
#### В корневой директории проекта нужно создать файл `.todoconfig` со следующим содержанием:
```
PREFIX=<префикс, после которого будет идти название задачи в файле>
SUFFIX=<расширение файлов, где искать задачи>
```
#### Также поиск задач будет производиться в файлах с расширением .todo
#### Формат задач:
```
<PREFIX> имя задачи (статус)
```
#### Опционально указывается статус (если не указать - статус задача открыта) в следующем формате:
```
[o] - открытая задача
[x] - закрытая задача
[*] - текущая задача
```
#### Запуск:
```
make
./todo (-a) - показать все задачи
./todo -o - показать открытые задачи
./todo -x - показать закрытые задачи
./todo -* - показать текущие задачи
```
